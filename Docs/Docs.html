Como instalar ubuntu server

Descarga la ISO de la web oficial de ubuntu

Cambia la ruta de configuración a ubuntu.cica

Siguiendo la instalación, crea un nuevo usuario y contraseña

Configurar el netplan

Edita el archivo de configuración ubicado en /etc/netplan

sudo nano /etc/netplan/"nombre del archivo"

Cambia la dirección IP a la de tu servidor

aplica el netplan con sudo netplan apply

Instalar apache2

ejecuta el comando sudo apt install apache2

Configuración de las direcciones web de tu servidor

Entra en /etc/apache2/sites-available

Haz copypaste del archivo 000-default , y cambiale el nombre al de tu dominio

edita el archivo con cualquier editor de texto e introduce tu IP en la primera linea del archivo (VirtualHost 192.168.5.56)

Ejecuta sudo service apache2 restart para reinciar el servicio y aplicar los cambios que has realizado

Instalar vsftpd

ejecutamos el comando sudo apt install vsftpd

entramos en /etc/vsftpd.conf y editamos el archivo

descomentamos las siguientes lineas

write_enable=YES

chroot_local_user=YES

local_root=/web

pasv_nin_port=40000

pasv_max_port=50000

userlist_enable=YES

userlist_file=/etc/vsftpd.userlist

userlist_deny=NO

ejecutamos el comando sudo service vsftpd restart para reiniciar el servicio y aplicar los cambios

Segundo servidor

Seguimos los mismos pasos para la instalación que con el servidor uno

volvemos a cambiar el netplan ubicado en la misma ruta

Instalamos bind9

Ejecutamos sudo apt install bind9

Instalación y configuración del DNS

nos dirigimos a la ruta /etc/bind

con un editor de texto editamos el archivo named.conf.local

cambiamos la estructura y la hacemos parecida a esta

copiamos el archivo y le cambiamos el nombre al que será el nombre de nuestra web

cp db.local db."tunombre".local

editamos el archivo y cambiamos la ip del servidor dos y la del dominio

Cliente

modificamos /etc/hosts

agregamos nuestra ip más el nombre de nuestro dominio

entramos al archivo /etc/resolv.conf

al final del archivo modificamos 127.0.0.53 por nuestra ip
